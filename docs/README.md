## Platform Overview

The Lily Pad is a comprehensive Web3 streaming and NFT launchpad platform built on the Monad blockchain. Key features include:

- **NFT Launchpad**: Create and launch NFT collections with multi-phase minting
- **Wallet Integration**: Seamless MetaMask connection with network management
- **Testnet Support**: Full testnet environment for testing before mainnet deployment
- **Live Streaming**: Integrated streaming capabilities for creators
- **Tipping System**: On-chain tipping for supporting creators
