# Quick Start Guide

Get started with The Lily Pad in under 5 minutes.

## Prerequisites

Before you begin, make sure you have:

- [ ] A Web3 wallet (MetaMask recommended)
- [ ] MON tokens (get testnet tokens from the [faucet](https://faucet.monad.xyz))
- [ ] A modern web browser (Chrome, Firefox, Brave)

## Step 1: Connect Your Wallet

1. Click the **Connect Wallet** button in the navigation bar
2. Select your wallet provider (MetaMask)
3. Approve the connection request

```
Tip: If you're new to Web3, we recommend starting on Testnet 
to practice without using real tokens.
```

## Step 2: Switch to Testnet (Recommended for New Users)

1. Look for the **Network Switch** in the navigation bar
2. Toggle the switch to enable **Testnet** mode
3. You'll see a yellow banner confirming you're on testnet

> ⚠️ You must disconnect your wallet before switching networks

## Step 3: Get Test Tokens

1. Click the **Faucet** button (visible when on testnet)
2. Visit the Monad Testnet Faucet
3. Enter your wallet address
4. Receive free test MON tokens

## Step 4: Explore the Launchpad

1. Navigate to the **Launchpad** page
2. Browse live, upcoming, and ended collections
3. Click on a collection to view details

## Step 5: Mint Your First NFT

1. Open a collection detail page
2. Check your wallet balance in the mint card
3. Select the number of NFTs to mint
4. Review the gas estimation
5. Click **Mint** and confirm the transaction

## Common Issues

### "Wrong Network" Error
Switch to the correct network (Mainnet or Testnet) using the network toggle.

### "Insufficient Balance" Error
Make sure you have enough MON to cover both the mint price and gas fees.

### Wallet Not Connecting
Try refreshing the page or disconnecting/reconnecting your wallet.
